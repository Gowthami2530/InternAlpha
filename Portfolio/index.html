MY SKILLS

HTML

HTML, or Hypertext Markup Language, is the standard language used to create and design web pages. It consists of a series of elements, which you use to enclose or wrap different parts of the content to make it appear or act a certain way.

CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

TAILWIND CSS

JAVASCRIPT

C

PYTHON
